# ハッシュアルゴリズムとは？

データの改ざん検知に使われる。
任意のデータ長から、固定長の文字列を生成する。

MD5, SHA-1, SHA-2, SHA-3

ブロックチェーン技術やSSH, TLS/SSLを支える技術である。

Dockerのコンテナレイヤに使用されていたり、WebpackのアセットURLに使用されていたりもする。

## SHA-2

SHA-1は過去に衝突したことがあり、信頼性なし。
よく使用されるのはSHA-2。
